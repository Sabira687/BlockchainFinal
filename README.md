# GameFi Protocol

A full-stack decentralized GameFi economy protocol built for the Blockchain Technologies 2 final project.

## Scenario

Option B — GameFi Economy.

## Core Features

- ERC-1155 in-game item economy
- Crafting system
- Constant-product AMM for fungible resources
- NFT rental vault
- Chainlink-compatible randomness and oracle modules
- DAO-governed game parameters
- ERC-4626 yield vault
- UUPS upgradeable configuration contract
- CREATE and CREATE2 factory deployment
- The Graph indexing
- L2 testnet deployment

## Tech Stack

- Solidity
- Foundry
- OpenZeppelin
- Chainlink
- React
- Wagmi
- Viem
- The Graph
